## Project Overview
This project is an ASP.NET MVC web application that simulates a Travel Agency system.
The project was developed as part of the Communication course and demonstrates
client-server communication, web request handling, and data flow between application layers.

## Project Objectives
- Develop a Travel Agency web system using ASP.NET MVC
- Demonstrate client-server communication principles
- Apply MVC architecture (Model, View, Controller)
- Handle HTTP requests and responses in a structured manner

## System Architecture
The system is implemented using the MVC design pattern:
- **Models**: Manage the application data and business logic
- **Views**: Responsible for the user interface
- **Controllers**: Handle user requests and coordinate between Models and Views

## Technologies Used
- Programming Language: C#
- Framework: ASP.NET MVC
- Development Environment: Visual Studio
- Version Control: Git & GitHub
